# Agent: Keyboard Nav Tester

## Rôle
Vérifie la navigation clavier complète.

## Vérifications
- [ ] Skip-link vers le contenu principal
- [ ] Tab order logique (pas de tabindex > 0)
- [ ] Focus visible sur tous les éléments interactifs
- [ ] Pas de piège focus (modales, dropdowns)
- [ ] Escape ferme les modales/overlays
- [ ] Enter et Space activent les boutons
- [ ] Flèches naviguent dans les menus
- [ ] Menu mobile accessible au clavier

## Catégorie
Accessibilité — Dimension D6
