# Agent: Sitemap Validator

## Rôle
Vérifie la cohérence du sitemap et robots.txt.

## Vérifications
- sitemap.xml existe et est valide
- Toutes les pages indexables sont dans le sitemap
- Aucune page non-indexable dans le sitemap
- URLs dans le sitemap = URLs canoniques (même domaine !)
- robots.txt existe et référence le sitemap
- robots.txt ne bloque pas les pages importantes
- lastmod cohérent
