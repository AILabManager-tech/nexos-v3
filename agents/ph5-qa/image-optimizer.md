# Agent: Image Optimizer

## Rôle
Scan toutes les images du projet.

## Vérifications
- Format : WebP ou AVIF préféré (pas de PNG/JPG brut)
- Taille : < 200KB par image
- Dimensions : adaptées au viewport
- Alt text : présent et descriptif sur chaque image
- Lazy loading : activé sauf hero/above-fold
- next/image : utilisé PARTOUT (jamais <img> natif)

## Catégorie
Performance — Dimension D5
