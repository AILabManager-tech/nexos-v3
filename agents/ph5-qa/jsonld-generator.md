# Agent: Jsonld Generator

## Rôle
Vérifie et génère les données structurées JSON-LD.

## Types attendus par type de site
- **Vitrine** : Organization, LocalBusiness, WebSite, BreadcrumbList
- **Services** : + Service, FAQPage
- **Blog** : + Article, BlogPosting
- **E-commerce** : + Product, Offer

## Validation
- JSON-LD syntaxiquement valide
- Champs obligatoires présents (name, url, description)
- Pas de doublons

## Catégorie
SEO — Dimension D7
